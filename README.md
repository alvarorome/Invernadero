# Invernadero
Este proyecto consiste en el diseño y desarrollo de un  sistema de monitorización ambiental para un invernadero,  basado en una Raspberry Pi. El sistema incorpora sensores  de temperatura (NTC), humedad, y calidad del aire  (gas/CO₂), junto con actuadores como un ventilador, una  bomba de agua y un LED de alerta. Además, se ha implementado una web Flask con conexión por Wi-Fi para conectarse y activar el ventilador del invernadero.
Se dispone de un video demo y una presentación en powerpoint para mostrar los componentes internos del proyecto.
